      23             : /*!
      24             :  *  \ingroup CoordinateMapsGroup
      25             :  *
      26             :  *  \brief Three dimensional map from the cube to a bulged cube.
      27             :  *  The cube is shaped such that the surface is compatible
      28             :  *  with the inner surface of Wedge3D.
      29             :  *  The shape of the object can be chosen to be cubical,
      30             :  *  if the sphericity is set to 0, or to a sphere, if
      31             :  *  the sphericity is set to 1. The sphericity can
      32             :  *  be set to any number between 0 and 1 for a bulged cube.
      33             :  *
      34             :  *  \details The volume map from the cube to a bulged cube is obtained by
      35             :  *  interpolating between six surface maps, twelve bounding curves, and
      36             :  *  eight corners. The surface map for the upper +z axis is obtained by
      37             :  *  interpolating between a cubical surface and a spherical surface. The
      38             :  *  two surfaces are chosen such that the latter circumscribes the former.
      39             :  *
      40             :  *  We make a choice here as to whether we wish to use the logical coordinates
      41             :  *  parameterizing these surface as they are, in which case we have the
      42             :  *  equidistant choice of coordinates, or whether to apply a tangent map to them
      43             :  *  which leads us to the equiangular choice of coordinates. In terms of the
      44             :  *  logical coordinates, the equiangular coordinates are:
      45             :  *
      46             :  *  \f[\textrm{equiangular xi} : \Xi(\xi) = \textrm{tan}(\xi\pi/4)\f]
      47             :  *
      48             :  *  \f[\textrm{equiangular eta}  : \mathrm{H}(\eta) = \textrm{tan}(\eta\pi/4)\f]
      49             :  *
      50             :  *  With derivatives:
      51             :  *
      52             :  *  \f[\Xi'(\xi) = \frac{\pi}{4}(1+\Xi^2)\f]
      53             :  *
      54             :  *  \f[\mathrm{H}'(\eta) = \frac{\pi}{4}(1+\mathrm{H}^2)\f]
      55             :  *
      56             :  *  The equidistant coordinates are:
      57             :  *
      58             :  *  \f[ \textrm{equidistant xi}  : \Xi = \xi\f]
      59             :  *
      60             :  *  \f[ \textrm{equidistant eta}  : \mathrm{H} = \eta\f]
      61             :  *
      62             :  *  with derivatives:
      63             :  *
      64             :  *  <center>\f$\Xi'(\xi) = 1\f$, and \f$\mathrm{H}'(\eta) = 1\f$</center>
      65             :  *
      66             :  *  We also define the variable \f$\rho\f$, given by:
      67             :  *
      68             :  *  \f[\rho = \sqrt{1+\Xi^2+\mathrm{H}^2}\f]
      69             :  *
      70             :  *  ### The Spherical Face Map
      71             :  *  The surface map for the spherical face of radius \f$R\f$ lying in the
      72             :  *  \f$+z\f$
      73             :  *  direction in either choice of coordinates is then given by:
      74             :  *
      75             :  *  \f[\vec{\sigma}_{spherical}: \vec{\xi} \rightarrow \vec{x}(\vec{\xi})\f]
      76             :  *  Where
      77             :  *  \f[
      78             :  *  \vec{x}(\xi,\eta) =
      79             :  *  \begin{bmatrix}
      80             :  *  x(\xi,\eta)\\
      81             :  *  y(\xi,\eta)\\
      82             :  *  z(\xi,\eta)\\
      83             :  *  \end{bmatrix}  = \frac{R}{\rho}
      84             :  *  \begin{bmatrix}
      85             :  *  \Xi\\
      86             :  *  \mathrm{H}\\
      87             :  *  1\\
      88             :  *  \end{bmatrix}
      89             :  *  \f]
      90             :  *
      91             :  *  ### The Cubical Face Map
      92             :  *  The surface map for the cubical face of side length \f$2L\f$ lying in the
      93             :  *  \f$+z\f$ direction is given by:
      94             :  *
      95             :  *  \f[\vec{\sigma}_{cubical}: \vec{\xi} \rightarrow \vec{x}(\vec{\xi})\f]
      96             :  *  Where
      97             :  *  \f[
      98             :  *  \vec{x}(\xi,\eta) =
      99             :  *  \begin{bmatrix}
     100             :  *  x(\xi,\eta)\\
     101             :  *  y(\xi,\eta)\\
     102             :  *  L\\
     103             :  *  \end{bmatrix}  = L
     104             :  *  \begin{bmatrix}
     105             :  *  \Xi\\
     106             :  *  \mathrm{H}\\
     107             :  *  1\\
     108             :  *  \end{bmatrix}
     109             :  *  \f]
     110             :  *
     111             :  *  ### The Bulged Face Map
     112             :  *  To construct the bulged map we interpolate between this cubical face map
     113             :  *  and a spherical face map of radius `radius_of_other_surface`, with the
     114             :  *  interpolation parameter being \f$s\f$, the `sphericity`.
     115             :  *  The surface map for the bulged face lying in the \f$+z\f$ direction is then
     116             :  *  given by:
     117             :  *
     118             :  *  \f[
     119             :  *  \vec{\sigma}_{+z}(\xi,\eta) = \left\{(1-s)L + \frac{sR}{\rho}\right\}
     120             :  *  \begin{bmatrix}
     121             :  *  \Xi\\
     122             :  *  \mathrm{H}\\
     123             :  *  1\\
     124             :  *  \end{bmatrix}
     125             :  *  \f]
     126             :  *
     127             :  *  We constrain L by demanding that the spherical face circumscribe the cube.
     128             :  *  With this condition, we have \f$L = R/\sqrt3\f$.
     129             :  *
     130             :  *  ### The General Formula for 3D Isoparametric Maps
     131             :  *  The general formula is given by Eq. 1 in section 2.1 of Hesthaven's paper
     132             :  *  "A Stable Penalty Method For The Compressible Navier-Stokes Equations III.
     133             :  *  Multidimensional Domain Decomposition Schemes" available
     134             :  *  <a href="
     135             :  *  http://citeseerx.ist.psu.edu/viewdoc/download?doi=10.1.1.699.1161&rep=rep1&type=pdf
     136             :  *  "> here </a>.
     137             :  *
     138             :  *  Hesthaven's formula is general in the degree of the shape functions used,
     139             :  *  so for our purposes we take the special case where the shape functions are
     140             :  *  linear, and define new variables accordingly.
     141             :  *
     142             :  *  We define the following variables for
     143             :  *  \f$\alpha, \beta, \gamma \in\{\xi,\eta,\zeta\}\f$:
     144             :  *
     145             :  *  \f[
     146             :  *  f^{\pm}_{\alpha} = \frac{1}{2}(1\pm\alpha)\\
     147             :  *  f^{\pm\pm}_{\alpha \ \beta} = \frac{1}{4}(1\pm\alpha)(1\pm\beta)\\
     148             :  *  f^{\pm\pm\pm}_{\alpha \ \beta \ \gamma} =
     149             :  *  \frac{1}{8}(1\pm\alpha)(1\pm\beta)(1\pm\gamma)
     150             :  *  \f]
     151             :  *
     152             :  *  The formula involves six surfaces, which we will denote by
     153             :  *  \f$\vec{\sigma}\f$, twelve curves, denoted by \f$\vec{\Gamma}\f$, and eight
     154             :  *  vertices, denoted by \f$\vec{\pi}\f$, with the subscripts denoting which
     155             :  *  face(s) these objects belong to. The full volume map is given by:
     156             :  *
     157             :  *  \f{align*}
     158             :  *  \phi(\xi,\eta,\zeta) = &
     159             :  *  f^{+}_{\zeta}\vec{\sigma}_{+\zeta}(\xi, \eta)+
     160             :  *  f^{-}_{\zeta}\vec{\sigma}_{-\zeta}(\xi, \eta)\\
     161             :  *  &+ f^{+}_{\eta}\vec{\sigma}_{+\eta}(\xi, \zeta)+
     162             :  *  f^{-}_{\eta}\vec{\sigma}_{-\eta}(\xi, \zeta)+
     163             :  *  f^{+}_{\xi}\vec{\sigma}_{+\xi}(\eta, \zeta)+
     164             :  *  f^{-}_{\xi}\vec{\sigma}_{-\xi}(\eta, \zeta)\\
     165             :  *  &- f^{++}_{\xi \ \eta}\vec{\Gamma}_{+\xi +\eta}(\zeta)-
     166             :  *  f^{-+}_{\xi \ \eta}\vec{\Gamma}_{-\xi +\eta}(\zeta)-
     167             :  *  f^{+-}_{\xi \ \eta}\vec{\Gamma}_{+\xi -\eta}(\zeta)-
     168             :  *  f^{--}_{\xi \ \eta}\vec{\Gamma}_{-\xi -\eta}(\zeta)\\
     169             :  *  &- f^{++}_{\xi \ \zeta}\vec{\Gamma}_{+\xi +\zeta}(\eta)-
     170             :  *  f^{-+}_{\xi \ \zeta}\vec{\Gamma}_{-\xi +\zeta}(\eta)-
     171             :  *  f^{+-}_{\xi \ \zeta}\vec{\Gamma}_{+\xi -\zeta}(\eta)-
     172             :  *  f^{--}_{\xi \ \zeta}\vec{\Gamma}_{-\xi -\zeta}(\eta)\\
     173             :  *  &- f^{++}_{\eta \ \zeta}\vec{\Gamma}_{+\eta +\zeta}(\xi)-
     174             :  *  f^{-+}_{\eta \ \zeta}\vec{\Gamma}_{-\eta +\zeta}(\xi)-
     175             :  *  f^{+-}_{\eta \ \zeta}\vec{\Gamma}_{+\eta -\zeta}(\xi)-
     176             :  *  f^{--}_{\eta \zeta}\vec{\Gamma}_{-\eta -\zeta}(\xi)\\
     177             :  *  &+ f^{+++}_{\xi \ \eta \ \zeta}\vec{\pi}_{+\xi +\eta +\zeta}+
     178             :  *  f^{-++}_{\xi \ \eta \ \zeta}\vec{\pi}_{-\xi +\eta +\zeta}+
     179             :  *  f^{+-+}_{\xi \ \eta \ \zeta}\vec{\pi}_{+\xi -\eta +\zeta}+
     180             :  *  f^{--+}_{\xi \ \eta \ \zeta}\vec{\pi}_{-\xi -\eta +\zeta}\\
     181             :  *  &+ f^{++-}_{\xi \ \eta \ \zeta}\vec{\pi}_{+\xi +\eta -\zeta}+
     182             :  *  f^{-+-}_{\xi \ \eta \ \zeta}\vec{\pi}_{-\xi +\eta -\zeta}+
     183             :  *  f^{+--}_{\xi \ \eta \ \zeta}\vec{\pi}_{+\xi -\eta -\zeta}+
     184             :  *  f^{---}_{\xi \ \eta \ \zeta}\vec{\pi}_{-\xi -\eta -\zeta}
     185             :  *  \f}
     186             :  *
     187             :  *
     188             :  *  ### The Special Case for Octahedral Symmetry
     189             :  *  The general formula is for the case in which there are six independently
     190             :  *  specified bounding surfaces. In our case, the surfaces are obtained by
     191             :  *  rotations and reflections of the upper-\f$\zeta\f$ face.
     192             :  *
     193             :  * We define the matrices corresponding to these transformations to be:
     194             :  *
     195             :  * \f[
     196             :  * S_{xy} =
     197             :  *  \begin{bmatrix}
     198             :  *  0 & 1 & 0\\
     199             :  *  1 & 0 & 0\\
     200             :  *  0 & 0 & 1\\
     201             :  *  \end{bmatrix},\
     202             :  *
     203             :  * S_{xz} =
     204             :  *  \begin{bmatrix}
     205             :  *  0 & 0 & 1\\
     206             :  *  0 & 1 & 0\\
     207             :  *  1 & 0 & 0\\
     208             :  *  \end{bmatrix},\
     209             :  *
     210             :  * S_{yz} =
     211             :  *  \begin{bmatrix}
     212             :  *  1 & 0 & 0\\
     213             :  *  0 & 0 & 1\\
     214             :  *  0 & 1 & 0\\
     215             :  *  \end{bmatrix}\f]
     216             :  *
     217             :  * \f[C_{zxy} =
     218             :  *  \begin{bmatrix}
     219             :  *  0 & 0 & 1\\
     220             :  *  1 & 0 & 0\\
     221             :  *  0 & 1 & 0\\
     222             :  *  \end{bmatrix},\
     223             :  *
     224             :  * C_{yzx} =
     225             :  *  \begin{bmatrix}
     226             :  *  0 & 1 & 0\\
     227             :  *  0 & 0 & 1\\
     228             :  *  1 & 0 & 0\\
     229             :  *  \end{bmatrix}\f]
     230             :  *
     231             :  * \f[N_{x} =
     232             :  *  \begin{bmatrix}
     233             :  *  -1 & 0 & 0\\
     234             :  *  0 & 1 & 0\\
     235             :  *  0 & 0 & 1\\
     236             :  *  \end{bmatrix},\
     237             :  *
     238             :  * N_{y} =
     239             :  *  \begin{bmatrix}
     240             :  *  1 & 0 & 0\\
     241             :  *  0 & -1 & 0\\
     242             :  *  0 & 0 & 1\\
     243             :  *  \end{bmatrix},\
     244             :  *
     245             :  * N_{z} =
     246             :  *  \begin{bmatrix}
     247             :  *  1 & 0 & 0\\
     248             :  *  0 & 1 & 0\\
     249             :  *  0 & 0 & -1\\
     250             :  *  \end{bmatrix}
     251             :  *  \f]
     252             :  *
     253             :  * The surface maps can now all be written in terms of
     254             :  * \f$\vec{\sigma}_{+\zeta}\f$ and these matrices:
     255             :  * <center>
     256             :  * \f$\vec{\sigma}_{-\zeta}(\xi, \eta) = N_z\vec{\sigma}_{+\zeta}(\xi, \eta)\\
     257             :  * \vec{\sigma}_{+\eta}(\xi, \zeta) = S_{yz}\vec{\sigma}_{+\zeta}(\xi, \zeta)\\
     258             :  * \vec{\sigma}_{-\eta}(\xi, \zeta) = N_yS_{yz}\vec{\sigma}_{+\zeta}(\xi,
     259             :  * \zeta)\\
     260             :  * \vec{\sigma}_{+\xi}(\eta, \zeta) = C_{zxy}\vec{\sigma}_{+\zeta}(\eta,
     261             :  * \zeta)\\
     262             :  * \vec{\sigma}_{-\xi}(\eta, \zeta) = N_xC_{zyx}\vec{\sigma}_{+\zeta}(\eta,
     263             :  * \zeta)\f$
     264             :  * </center>
     265             :  *
     266             :  * The four bounding curves \f$\vec{\Gamma}\f$ on the \f$+\zeta\f$ face are
     267             :  * given by:
     268             :  *
     269             :  * <center>
     270             :  * \f$\vec{\Gamma}_{+\xi,+\zeta}(\eta) = \vec{\sigma}_{+\zeta}(+1,\eta)\\
     271             :  * \vec{\Gamma}_{-\xi,+\zeta}(\eta) = \vec{\sigma}_{+\zeta}(-1,\eta)
     272             :  * = N_x\vec{\sigma}_{+\zeta}(+1, \eta)\\
     273             :  * \vec{\Gamma}_{+\eta,+\zeta}(\xi) = \vec{\sigma}_{+\zeta}(\xi,+1)
     274             :  * = S_{xy}\vec{\sigma}_{+\zeta}(+1, \xi)\\
     275             :  * \vec{\Gamma}_{-\eta,+\zeta}(\xi) = \vec{\sigma}_{+\zeta}(\xi,-1)
     276             :  * = N_yS_{xy}\vec{\sigma}_{+\zeta}(+1,\xi)\f$
     277             :  * </center>
     278             :  *
     279             :  * The bounding curves on the other surfaces can be obtained by transformations
     280             :  * on the \f$+\zeta\f$ face:
     281             :  *
     282             :  * <center>
     283             :  * \f$\vec{\Gamma}_{+\xi,-\zeta}(\eta) = N_z\vec{\sigma}_{+\zeta}(+1,\eta)\\
     284             :  * \vec{\Gamma}_{-\xi,-\zeta}(\eta) = N_z\vec{\sigma}_{+\zeta}(-1,\eta)
     285             :  * = N_zN_x\vec{\sigma}_{+\zeta}(+1,\eta)\\
     286             :  * \vec{\Gamma}_{+\eta,-\zeta}(\xi) = N_z\vec{\sigma}_{+\zeta}(\xi,+1)
     287             :  * = N_zS_{xy}\vec{\sigma}_{+\zeta}(+1, \xi)\\
     288             :  * \vec{\Gamma}_{-\eta,-\zeta}(\xi) = N_z\vec{\sigma}_{+\zeta}(\xi,-1)
     289             :  * = N_zN_yS_{xy}\vec{\sigma}_{+\zeta}(+1, \xi)\\
     290             :  * \vec{\Gamma}_{+\xi,+\eta}(\zeta) =
     291             :  * C_{zxy}\vec{\sigma}_{+\zeta}(+1,\zeta)\\
     292             :  * \vec{\Gamma}_{-\xi,+\eta}(\zeta) =
     293             :  * N_xC_{zxy}\vec{\sigma}_{+\zeta}(+1,\zeta)\\
     294             :  * \vec{\Gamma}_{+\xi,-\eta}(\zeta) = C_{zxy}\vec{\sigma}_{+\zeta}(-1,\zeta)
     295             :  * = C_{zxy}N_x\vec{\sigma}_{+\zeta}(+1,\zeta)\\
     296             :  * \vec{\Gamma}_{-\xi,-\eta}(\zeta) = N_xC_{zxy}\vec{\sigma}_{+\zeta}(-1,\zeta)
     297             :  * = N_xC_{zxy}N_x\vec{\sigma}_{+\zeta}(+1,\zeta)\f$
     298             :  * </center>
     299             :  *
     300             :  * Now we can write the volume map \f$\phi\f$ in terms of
     301             :  * \f$\vec{\sigma}_{+\zeta}\f$ only:
     302             :  * \f{align*}\phi(\xi,\eta,\zeta) = &
     303             :  * (f^{+}_{\zeta} + f^{-}_{\zeta}N_z)
     304             :  * \vec{\sigma}_{+\zeta}(\xi, \eta)\\
     305             :  * &+ (f^{+}_{\eta} + f^{-}_{\eta}N_y)
     306             :  * S_{yz}\vec{\sigma}_{+\zeta}(\xi, \zeta)\\
     307             :  * &+ (f^{+}_{\xi} + f^{-}_{\xi}N_x)
     308             :  * C_{zxy}\vec{\sigma}_{+\zeta}(\eta, \zeta)\\
     309             :  * &- (f^{+}_{\xi}+f^{-}_{\xi}N_x)
     310             :  * (f^{+}_{\eta}+f^{-}_{\eta}N_y)
     311             :  * C_{zxy}\vec{\sigma}_{+\zeta}(+1, \zeta)\\
     312             :  * &- (f^{+}_{\zeta}+f^{-}_{\zeta}N_z)\left\{
     313             :  * (f^{+}_{\xi}+f^{-}_{\xi}N_x)\vec{\sigma}_{+\zeta}(+1, \eta)+
     314             :  * (f^{+}_{\eta}+f^{-}_{\eta}N_y)S_{xy}\vec{\sigma}_{+\zeta}(+1, \xi)\right\}\\
     315             :  * &+ \frac{r}{\sqrt{3}}\vec{\xi}
     316             :  *  \f}
     317             :  *
     318             :  * Note that we can now absorb all of the \f$f\f$s into the matrix prefactors
     319             :  * in the above equation and obtain a final set of matrices. We define the
     320             :  * following *blending matrices*:
     321             :  *
     322             :  *  \f[
     323             :  *  B_{\xi} =
     324             :  *  \begin{bmatrix}
     325             :  *  0 & 0 & \xi\\
     326             :  *  1 & 0 & 0\\
     327             :  *  0 & 1 & 0\\
     328             :  *  \end{bmatrix},\
     329             :  *
     330             :  *  B_{\eta} =
     331             :  *  \begin{bmatrix}
     332             :  *  1 & 0 & 0\\
     333             :  *  0 & 0 & \eta\\
     334             :  *  0 & 1 & 0\\
     335             :  *  \end{bmatrix},\
     336             :  *
     337             :  *  B_{\zeta} =
     338             :  *  \begin{bmatrix}
     339             :  *  1 & 0 & 0\\
     340             :  *  0 & 1 & 0\\
     341             :  *  0 & 0 & \zeta\\
     342             :  *  \end{bmatrix}\\
     343             :  *
     344             :  *  B_{\xi\eta} =
     345             :  *  \begin{bmatrix}
     346             :  *  0 & 0 & \xi\\
     347             :  *  \eta & 0 & 0\\
     348             :  *  0 & 1 & 0\\
     349             :  *  \end{bmatrix},\
     350             :  *
     351             :  *  B_{\xi\zeta} =
     352             :  *  \begin{bmatrix}
     353             :  *  \xi & 0 & 0\\
     354             :  *  0 & 1 & 0\\
     355             :  *  0 & 0 & \zeta\\
     356             :  *  \end{bmatrix},\
     357             :  *
     358             :  *  B_{\eta\zeta} =
     359             :  *  \begin{bmatrix}
     360             :  *  0 & 1 & 0\\
     361             :  *  \eta & 0 & 0\\
     362             :  *  0 & 0 & \zeta\\
     363             :  *  \end{bmatrix}\\
     364             :  *
     365             :  *  B_{\xi\eta\zeta} =
     366             :  *  \begin{bmatrix}
     367             :  *  \xi & 0 & 0\\
     368             :  *  0 & \eta & 0\\
     369             :  *  0 & 0 & \zeta\\
     370             :  *  \end{bmatrix}
     371             :  *  \f]
     372             :  *
     373             :  *  Now we can write the volume map \f$\phi\f$ in these terms:
     374             :  *
     375             :  * \f{align*}
     376             :  * \phi(\xi,\eta,\zeta) = &
     377             :  * B_{\zeta}
     378             :  * \vec{\sigma}_{+\zeta}(\xi, \eta)\\& +
     379             :  * B_{\eta}
     380             :  * \vec{\sigma}_{+\zeta}(\xi, \zeta)+
     381             :  * B_{\xi}
     382             :  * \vec{\sigma}_{+\zeta}(\eta, \zeta)\\& -
     383             :  * B_{\xi \eta}
     384             :  * \vec{\sigma}_{+\zeta}(+1, \zeta)-
     385             :  * B_{\xi \zeta}
     386             :  * \vec{\sigma}_{+\zeta}(+1, \eta)+
     387             :  * B_{\eta \zeta}
     388             :  * \vec{\sigma}_{+\zeta}(+1, \xi)\\& +
     389             :  * B_{\xi \eta \zeta}
     390             :  * \vec{\sigma}_{+\zeta}(+1, +1)
     391             :  * \f}
     392             :  *
     393             :  * ### The Bulged Cube Map
     394             :  * We now use the result above to provide the mapping for the bulged cube.
     395             :  * First we will define the variables \f$\rho_A\f$ and \f$\rho_{AB}\f$, for
     396             :  * \f$A, B \in \{\Xi,\mathrm{H}, \mathrm{Z}\} \f$:
     397             :  * \f[
     398             :  * \rho_A = \sqrt{2 + A^2}\\
     399             :  * \rho_{AB} = \sqrt{1 + A^2 + B^2}
     400             :  * \f]
     401             :  * The final mapping is then:
     402             :  * \f[
     403             :  * \vec{x}(\xi,\eta,\zeta) = \frac{(1-s)r}{\sqrt{3}}
     404             :  * \begin{bmatrix}
     405             :  * \Xi\\
     406             :  * \mathrm{H}\\
     407             :  * \mathrm{Z}\\
     408             :  * \end{bmatrix} +
     409             :  * \frac{sr}{\sqrt{3}}
     410             :  * \begin{bmatrix}
     411             :  * \xi\\
     412             :  * \eta\\
     413             :  * \zeta\\
     414             :  * \end{bmatrix} + sr
     415             :  * \begin{bmatrix}
     416             :  * \xi & \Xi & \Xi\\
     417             :  * \mathrm{H} & \eta &\mathrm{H}\\
     418             :  * \mathrm{Z} & \mathrm{Z} & \zeta\\
     419             :  * \end{bmatrix}
     420             :  * \begin{bmatrix}
     421             :  * 1/\rho_{\mathrm{H}\mathrm{Z}}\\
     422             :  * 1/\rho_{\Xi\mathrm{Z}}\\
     423             :  * 1/\rho_{\Xi\mathrm{H}}\\
     424             :  * \end{bmatrix} - sr
     425             :  * \begin{bmatrix}
     426             :  * \Xi & \xi & \xi\\
     427             :  * \eta & \mathrm{H} &\eta\\
     428             :  * \zeta & \zeta & \mathrm{Z}\\
     429             :  * \end{bmatrix}
     430             :  * \begin{bmatrix}
     431             :  * 1/\rho_{\Xi}\\
     432             :  * 1/\rho_{\mathrm{H}}\\
     433             :  * 1/\rho_{\mathrm{Z}}\\
     434             :  * \end{bmatrix}
     435             :  * \f]
     436             :  *
     437             :  * In the case where the same coordinates are used for the cube and the sphere,
     438             :  * we have \f$\xi = \Xi\f$, etc. In this case, the formula reduces further.
     439             :  * It is given by:
     440             :  *
     441             :  * \f[
     442             :  * \vec{x}(\xi,\eta,\zeta) =
     443             :  * \left\{
     444             :  * \frac{r}{\sqrt{3}}
     445             :  * + sr
