37 /// \ingroup DataBoxTagsGroup
38 /// \brief Prefix indicating the divergence
39 ///
40 /// There are two variants of this tag that change how the derivatives are
41 /// computed depending on which is chosen. The simplest is the non-compute
42 /// item versions for a Tensor tag. This takes two template parameters:
43 /// 1. The tag to wrap
44 /// 2. The frame in which the divergence was taken
45 ///
46 /// The second variant of the divergence tag is a compute item that computes
47 /// the divergence in a Frame that is not the logical frame. This compute
48 /// item is used by specifying the the list of flux tags to be differentiated,
49 /// and the Tag for the inverse Jacobian between the logical frame and the frame
50 /// in which the divergence is taken.
51 template <typename T, typename S>
52 struct div : Tags_detail::div_impl<T, S> {};
53 } // namespace Tags
